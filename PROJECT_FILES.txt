# Trip Visualizer - Project Files

## Core Application Files

### Python Application
- **app.py** (81K) - Main Streamlit application with UI and all features
- **text_parser.py** (12K) - Parses structured text format into internal JSON
- **travel_extractor.py** (42K) - Gmail API integration for extracting bookings
- **export_pdf_functions.py** (16K) - PDF generation with map screenshots
- **api.py** (4K) - API endpoints

### Data Files
- **itinerary.json** (15K) - Demo data (Thailand trip sample)
- **sample_text_itinerary.txt** (663B) - Example of text format

### Configuration
- **credentials.json** (405B) - Gmail API OAuth credentials
- **token.json** (743B) - Gmail API auth token (auto-generated)
- **config.example.json** (265B) - Example configuration
- **requirements.txt** (376B) - Python dependencies

### User Documentation
- **README.md** (4K) - Developer/technical documentation
- **README.txt** (8.9K) - Complete user guide with instructions
- **QUICK_START.txt** (8K) - Quick reference card for users
- **AI_TEXT_CONVERSION_PROMPT.txt** (5.6K) - AI prompt for structuring itineraries

### Scripts
- **start.sh** (725B) - Shell script to start the application
