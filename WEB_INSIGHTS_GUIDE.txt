================================================================================
          WEB-BASED INSIGHTS - GET REAL-TIME TRAVEL INFORMATION
================================================================================

The Trip Visualizer can now fetch REAL-TIME travel insights from the web,
including current weather, prices, opening hours, and local tips!

================================================================================
                        HOW TO USE WEB INSIGHTS
================================================================================

OPTION 1: MANUAL (Recommended - Free & Easy)
────────────────────────────────────────────

1. Generate your itinerary in Trip Visualizer
2. Click "🌐 Get Real-Time Insights" button
3. Copy the generated prompt
4. Paste into ChatGPT (gpt.openai.com) or Claude (claude.ai)
5. Wait for AI to generate insights JSON
6. Copy the JSON response
7. Paste back into Trip Visualizer
8. Click "✅ Load Custom Insights"
9. Your insights are now live and up-to-date!

OPTION 2: AUTOMATIC (Requires OpenAI API Key)
────────────────────────────────────────────

Coming soon - will automatically fetch insights without manual steps.

================================================================================
                        TIPS FOR BEST RESULTS
================================================================================

✓ Use ChatGPT-4 or Claude Opus/Sonnet for best quality
✓ If response is incomplete, ask "continue" or "show more"
✓ Verify the JSON is complete (starts with [ and ends with ])
✓ Check for proper formatting before pasting back
✓ If error occurs, re-copy just the JSON array (remove any extra text)

COMMON ISSUES:
❌ AI adds explanation text → Copy only the JSON array part
❌ AI uses markdown ``` tags → Remove the ``` before and after
❌ JSON is cut off → Ask AI to continue from where it stopped
❌ Missing quotes or commas → Use jsonlint.com to validate

================================================================================
                        SWITCHING BETWEEN MODES
================================================================================

DEFAULT MODE (Static Insights):
- Uses pre-programmed data from insights_generator.py
- Instant loading, no internet required
- Good for common destinations
- Weather is seasonal averages

WEB MODE (Dynamic Insights):
- Uses AI-generated real-time data
- Requires internet and AI access
- Works for ANY destination
- Weather is current forecast

SWITCHING:
- Click "Get Real-Time Insights" → Switches to Web Mode
- Click "Use Default Insights" → Switches back to Static Mode
- Custom insights persist until you switch back

================================================================================
                        TROUBLESHOOTING
================================================================================

ISSUE: Button not visible
SOLUTION: Scroll down to "Key Insights" section after generating itinerary

ISSUE: Prompt too long for AI
SOLUTION: Split trip into segments (first week, second week)

ISSUE: JSON validation fails
SOLUTION:
  1. Go to jsonlint.com
  2. Paste the AI response
  3. Fix any syntax errors
  4. Copy the corrected JSON

ISSUE: Some insights irrelevant
SOLUTION: Edit the JSON manually - remove unwanted insights

ISSUE: Want more insights
SOLUTION: After loading, ask AI "Generate 10 more insights" with same trip details
